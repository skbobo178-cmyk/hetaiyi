## Crypto Price Tracker

即時追蹤加密貨幣價格

## Usage

```bash
python3 crypto_price.py [coin1,coin2]
```

## Examples

```bash
# Default coins (BTC, ETH, SOL, etc)
python3 crypto_price.py

# Custom coins
python3 crypto_price.py btc,eth,sol,avax
python3 crypto_price.py dogecoin,shiba-inu
```

## Features

- Real-time prices from CoinGecko
- 24h price change
- Emoji trends

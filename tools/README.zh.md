# Web Toolkit 🛠️

單一 HTML 檔案包含多種日常工具。無需安裝，瀏覽器打開就能用！

[English](README.md) | [中文](README.zh.md)

---

## 功能

| 工具 | 說明 |
|------|------|
| 🔗 URL 清理器 | 移除網址追蹤參數 |
| 🖼️ 圖片轉 Base64 | 將圖片轉為 HTML 可用編碼 |
| 📋 JSON 格式化 | 美化或壓縮 JSON |
| ⏰ 時間戳轉換 | 時間戳與日期互轉 |
| 🎨 顏色轉換器 | HEX、RGB、HSL 互轉 |
| 📱 QR Code 產生器 | 從文字/網址產生 QR Code |

## 使用方式

**直接用瀏覽器打開 HTML 檔案：**
```bash
# 雙擊打開
open webtoolkit.html

# 或拖曳到瀏覽器
```

無需伺服器、無需安裝、無需依賴！
